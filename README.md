\# Paradise Nursery Shopping Application



\## Project Name

Paradise Nursery Shopping Application



\## Description

Paradise Nursery is a shopping cart application for an online plant shop. The application allows users to browse different houseplants, view plant details, add plants to a shopping cart, update item quantities, delete items from the cart, and view the total cart cost.



\## Features

\- Landing page with company name and Get Started button

\- Product listing page with houseplants

\- Plant cards with image, name, description, price, and Add to Cart button

\- Plants grouped into categories

\- Shopping cart page

\- Increase and decrease quantity buttons

\- Delete item button

\- Continue Shopping button

\- Checkout button showing a Coming Soon message

\- Dynamic cart quantity in the navigation bar



\## Technologies Used

\- React

\- Redux Toolkit

\- JavaScript

\- CSS

\- GitHub Pages
